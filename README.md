# Mercedes-AMG PETRONAS F1 Team — Fan Concept

A responsive fan-made Formula 1 team website concept inspired by the Mercedes-AMG PETRONAS Formula One Team.

## Live Demo
[View the live website](https://firecat00.github.io/mercedesf1-fan-concept/)

## Features
- Responsive navigation
- Hero section with slider controls
- Featured stories and latest-news layout
- Driver profiles
- Merchandise section
- Interactive fan experiences
- Newsletter interface
- External links to relevant official pages

## Built With
- HTML5
- CSS3
- JavaScript
